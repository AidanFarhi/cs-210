# Chada Tech Clocks

This program enables the user to display and manipulate a 12 and 24 hour clock.

### Example

Menu:

```
**************************
* 1 - Add One Hour       *
* 2 - Add One Minute     *
* 3 - Add One Second     *
* 4 - Exit Program       *
**************************
```

Clock display:

```
**************************   **************************
*     12-Hour Clock      *   *     24-Hour Clock      *
*      12:12:12 PM       *   *        12:12:12        *
**************************   **************************
```

### To compile and run program (Mac/Linux)

`g++ -o main.out main.cpp`

`./main.out`
